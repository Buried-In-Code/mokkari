=======
Mokkari
=======

Quick Description
-----------------
A python wrapper for the metron.cloud_ API.

.. _metron.cloud: https://metron.cloud

Installation
------------

PyPi
~~~~

.. code:: bash

  $ pip3 install --user mokkari
